# 國中在校成績證明書調整（0417）

## 調整項目

* 修正 `PrintForm_StudentScoreCertificattion.cs` 原始成績來源（中文版）

## 問題原因

* 原本 `_原始成績` / `_原始等第` 直接使用 `subjectscore.Value.Score`
* `SubjectScore` 未保證以屬性方式暴露原始成績欄位，改由學期成績 XML 的 `Subject` 節點讀取 `原始成績` 屬性

## 修正內容

* 優先從學期成績 XML 的 `Subject` 節點讀取 `原始成績` 屬性（`K12.Data.Decimal.ParseAllowNull`）
* 依 `領域`、`科目` 對應節點；並嘗試 `ScoreInfo/SemesterSubjectScoreInfo/Subject` 與 `Field/ScoreInfo/SemesterSubjectScoreInfo/Subject` 兩種相對路徑
* 從學期成績紀錄物件中以反射取得內嵌之 `XmlElement` 根節點（僅供讀取，不變更既有成績邏輯）
* 若無法從 XML 取得原始成績，則後援使用 `K12.Data.SubjectScore.ScoreOrigin`（與 `PrintForm_StudentScoreCertificattion_English.cs` 一致）
* 補上空值防護：仍無原始成績時 `_原始成績` 為 null、`_原始等第` 為空字串

## 修正結果

* 成績與原始成績已分離
* 無資料時不會拋錯
* 報表可正常輸出

## 實作檔案

* `PrintForm_StudentScoreCertificattion.cs`（中文版）
* `PrintForm_StudentScoreCertificattion_English.cs`（英文版以 `ScoreOrigin` 為主；中文版多 XML 路徑與後援）
