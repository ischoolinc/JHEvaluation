# 🎯 目標

1. **僅調整英文版**

   * 語文_科目 合併欄位由 1~6 → 支援到 1~10
2. **合併欄位總表分開產生**

   * 英文版、中文版輸出不同檔案
   * 中文版邏輯完全不變（不可影響）
3. **畫面呼叫方式不變**

   * 保留：Global.ExportMappingFieldWord(string doc)

---

# 📍 修改檔案

* Global.cs

---

# ✅ Step 1：新增英文版專用常數

📍位置：class Global 內

```csharp
const int EnglishDomainSubjectMax = 10;
const int DefaultDomainSubjectMax = 6;
```

---

# ✅ Step 2：輸出檔名分流（中英文分開）

📍搜尋：

```csharp
string inputReportName = "在校成績證明書合併欄位總表";
```

📍修改為：

```csharp
string inputReportName = "在校成績證明書合併欄位總表_中文版";

if (doc == "英文版")
{
    inputReportName = "在校成績證明書合併欄位總表_英文版";
}
```

📌 目的：

* 中文 / 英文 產生不同檔案
* 不影響 UI 呼叫

---

# ✅ Step 3：建立科目數上限控制變數

📍在方法內（ExportMappingFieldWord）加入：

```csharp
int subjectMax = DefaultDomainSubjectMax;

if (doc == "英文版")
{
    subjectMax = EnglishDomainSubjectMax;
}
```

---

# ✅ Step 4：修改「學期科目成績」區塊

📍搜尋：

```csharp
for (int i = 1; i <= 6; i++)
```

📍修改為：

```csharp
for (int i = 1; i <= subjectMax; i++)
```

📌 影響欄位（英文版）：

* 語文_科目名稱7~10
* 語文_科目7~10_權數1~6
* 語文_科目7~10_成績1~6
* 語文_科目7~10_等第1~6
* 語文_科目7~10_平均成績
* 語文_科目7~10_平均成績等第

📌 中文版：
👉 仍維持 1~6（因 subjectMax = 6）

---

# ✅ Step 5：修改「學期科目原始成績」區塊

📍搜尋：

```csharp
for (int i = 1; i <= 6; i++)
```

📍修改為：

```csharp
for (int i = 1; i <= subjectMax; i++)
```

📌 影響欄位（英文版）：

* 語文_科目名稱7~10
* 語文_科目7~10_原始成績1~6
* 語文_科目7~10_原始等第1~6

📌 中文版：
👉 不受影響

---

# ⚠️ Step 6：不要修改以下區塊

```csharp
if (domain == "彈性課程")
    for (int i = 7; i <= 18; i++)
```

* 與語文調整無關
