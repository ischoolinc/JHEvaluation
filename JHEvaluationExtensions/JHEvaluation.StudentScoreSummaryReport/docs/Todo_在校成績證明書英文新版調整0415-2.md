# ✅ 問題原因整理

目前 `table.Columns` 是依 `DomainList` 建立領域欄位。
但後面在整理 `jssr.Domains` 平均成績時，會依實際 `domainscore.Value.Domain` 動態產生 key。

當實際資料有：

* `彈性學習課程`

但 `DataTable` 沒有建立：

* `領域_彈性學習課程_平均成績`

最後執行：

```csharp
row[key] = domainScore_dict[key];
```

就會拋出：

* `資料行 '領域_彈性學習課程_平均成績' 不屬於資料表。`
