# Todo: 在合併欄位總表（Global.ExportMappingFieldWord）加入三個新欄位

## 目標
在 `Global.cs` 的 `ExportMappingFieldWord(string doc)` 方法中，於輸出「合併欄位總表 Word」時，新增下列三個 **固定欄位**（以表格呈現並插入 MERGEFIELD）：
- **出生日期民國年**
- **入學日期民國年**
- **畢業日期民國年**

> 這三個欄位對應 `PrintForm_StudentScoreCertificattion.cs` 中 DataTable 已填入的欄位值，格式為：`民國YYY年M月D日`（例：民國104年9月9日）。

## Todo 1：在 ExportMappingFieldWord 增加「基本資料（日期）」區塊
**位置建議**：在既有的「缺曠動態產生合併欄位」與「學期領域成績」區塊之間，加入以下程式碼區塊（或放在方法尾端、存檔前亦可）。

```csharp
// === 新增：基本資料（日期）欄位 ===
builder.Writeln();
builder.Writeln();
builder.Writeln("基本資料（日期）");
builder.StartTable();

// 欄位：出生日期民國年
builder.InsertCell();
builder.Write("出生日期民國年");
builder.InsertCell();
builder.InsertField("MERGEFIELD 出生日期民國年 \\* MERGEFORMAT ", "«出生日期民國年»");
builder.EndRow();

// 欄位：入學日期民國年
builder.InsertCell();
builder.Write("入學日期民國年");
builder.InsertCell();
builder.InsertField("MERGEFIELD 入學日期民國年 \\* MERGEFORMAT ", "«入學日期民國年»");
builder.EndRow();

// 欄位：畢業日期民國年
builder.InsertCell();
builder.Write("畢業日期民國年");
builder.InsertCell();
builder.InsertField("MERGEFIELD 畢業日期民國年 \\* MERGEFORMAT ", "«畢業日期民國年»");
builder.EndRow();

builder.EndTable();
// === 基本資料（日期）欄位：結束 ===
```

> **說明**
> - 使用 `DocumentBuilder` 的 `InsertField("MERGEFIELD ...")` 插入三個固定欄位，名稱需與 MailMerge 欄位一致：
>   - `出生日期民國年`
>   - `入學日期民國年`
>   - `畢業日期民國年`
> - 這些欄位為**固定欄位**，非動態迴圈產生，不受 `doc`（中文版/英文版）參數影響。若未來英文版需要不同顯示，可另行在 `doc == "英文版"` 分支加註英文列名，但 **MERGEFIELD 名稱應保持中文** 以對應 `DataTable` 欄位。

## 驗收項目
- 執行「下載合併欄位總表」功能（`Global.ExportMappingFieldWord("中文版")`），輸出的 Word 內應多一段「基本資料（日期）」表格，並包含三列：
  - 出生日期民國年 → «出生日期民國年»
  - 入學日期民國年 → «入學日期民國年»
  - 畢業日期民國年 → «畢業日期民國年»
- 使用實際樣板進行 MailMerge 測試，三個欄位能正確帶出資料（例：民國104年9月9日）。
